# Distance to Coast - Welcome!
---

### Overview:
The distance to coast API allows you to determine the distance between:
 - A latitude / longitude geographic point
 - An address

The API returns the top 3 calculated closest coastlines from the requested address or geographic point.

### API Options
Along with the distance calculation, you may request the following options along with your request.

 - Elevation (ele T/F) - elevation of the requested address or geographic point
 - Line (lines T/F) - return the line GeoJSON
 - Coasts (coast T/F) - return the intersected coastline GeoJSON

### Payload Options

Address based payload:

`{"address":"6520 S 15th Ave, Richfield, MN 55423","lines":true, "ele":true, "coast":true} `

Geographic point based payload:

`{"lat": 37.345, "lon": -77.7654, "lines":true, "ele":true, "coast":false}`
